# Shooter Simple

یک بازی ساده زامبی‌کشی ساخته شده با **Pygame**. هدف بازی این است که تا جایی که می‌توانید زامبی‌ها را بکشید و از سلامت خود محافظت کنید.

---

## ویژگی‌ها

- **بازیکن** با کنترل کیبورد حرکت می‌کند.
- **چند حالت شلیک**:
  1. تیر تک
  2. تیر دوبل
  3. تیر سه‌تایی
  4. موشک با انفجار منطقه‌ای
- **زامبی‌ها** به سمت بازیکن حرکت می‌کنند.
- **امتیاز و موج‌ها**:
  - هر موج شامل تعدادی زامبی است که پس از پاک شدن، موج بعدی با تعداد زامبی‌های بیشتر ظاهر می‌شود.
- **نوار سلامتی** و نمایش امتیاز و حالت شلیک.

---

## کنترل‌ها

- **حرکت بازیکن:**  
  - `W` = بالا  
  - `S` = پایین  
  - `A` = چپ  
  - `D` = راست
- **شلیک:** `SPACE`  
- **تغییر حالت شلیک:** `Q`  
- **خروج از بازی:** `ESC` (در صفحه پایان بازی)

---
